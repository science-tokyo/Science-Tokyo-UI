# タイポグラフィー Typography

使用する文字サイズは、定義された範囲内で使用します。  
統一されたタイポグラフィを維持するため、規定外のサイズは使用を避けてください。  
文字サイズのルールに加え、html タグの lang 属性が ja の場合は、Noto Sans JP が適用され、  en の場合は Hind フォントが適用されます。

## 太さ｜フォントウェイト

文字ウェイトは以下の中のいずれかを使用します。  
sass:mapで定義しています。`$font-weight`のキーに適用したいウェイトを指定します。

```scss
font-weight: map.get($font-weight, thin);
```

## 基本文字スタイル

文字スタイルは、文字サイズ（font-size）及び行間（line-height）、文字間（letter-spacing）、文字の太さ（font-weight）はセットで定義され、以下の種類が定義されます。  
Mixinを使用してスタイルを適用します。各スタイルの詳細は`_typography.scss`を参照してください。

```scss
@include txt($t-displayL);
```
