# サイズ Size

サイトにおける 要素のサイズを定義します。  
ただし、縦横比が固定された画像やビデオ、グリッドに沿って設定される要素は対象外です。  
以下のCSS変数でサイズを設定できます。詳細は`_var_numbers.scss`を参照してください。
