# 角丸 Corner radius

要素に対する角丸を定義します。角丸は視覚的に独立した印象を与えることができます。  
要素のサイズ、内包する要素との間隔に応じて適切な半径を設定してください。  
以下のCSS変数で角丸を設定できます。詳細は`_var_numbers.scss`を参照してください。

<ul class="docs-size">
  <li class="docs-size_radius-sss">`--semantic-corner-radius-sss`: 1px</li>
  <li class="docs-size_radius-ss">`--semantic-corner-radius-ss`: 2px</li>
  <li class="docs-size_radius-s">`--semantic-corner-radius-s`: 4px</li>
  <li class="docs-size_radius-m">`--semantic-corner-radius-m`: 6px</li>
  <li class="docs-size_radius-ms">`--semantic-corner-radius-ms`: 8px</li>
  <li class="docs-size_radius-ml">`--semantic-corner-radius-ml`: 10px</li>
  <li class="docs-size_radius-l">`--semantic-corner-radius-l`: 12px</li>
  <li class="docs-size_radius-ll">
    `--semantic-corner-radius-ll`: 1rem（16px）
  </li>
  <li class="docs-size_radius-lll">`--semantic-corner-radius-lll`: 32px</li>
  <li class="docs-size_radius-pill">`--semantic-corner-radius-pill`: 9999px</li>
</ul>
