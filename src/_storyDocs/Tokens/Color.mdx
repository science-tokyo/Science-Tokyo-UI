# 色 Color

色はプリミティブトークンから定義され、直接UIに適用されるのではなく、セマンティックトークンをバリアブルを通じて利用します。  
詳細は`_var_color.scss`を参照してください。

## プリミティブトークン／Primitive token

サイトで使用可能な配色の全貌です。UI（コンポーネント）に直接この色が設定されることはありません。  
セマンティックトークンとして定義された値をコンポーネントに適用します。

## セマンティックトークン／Semantic token

UI（コンポーネント）に適用するために設定した配色です。インタラクションや背景色などにも利用しています。  
ブランドカラーに加えて、9色のカラーで構成しています。Primary／Secondary／Tertiary／Quaternaryは、組み合わせやコントラスト比などを考慮した場合に利用可能な優先度のバリエーションです。
