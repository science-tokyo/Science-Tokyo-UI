# Science Tokyo UI

## 概要

本ドキュメントは、Science Tokyo UI フロントエンドリソースを利用するためのガイドです。  
この環境は、主に`dist`配下に生成されたファイルを利用者が複製し、個別サイトの実装基盤として利用することを想定しています。  
多様なケースに対応するため、現時点では特定のフレームワークやCMSに依存しない、HTML/CSS/JavaScriptのみで構成されています。

### 想定される利用シナリオ

- GitHubリポジトリをクローン
- 利用シナリオ
  1. **そのまま利用**
     - dist配下のファイルを複製してサイト実装
  2. **カスタマイズして利用**:
     - デザイントークン（CSS変数）の調整
     - 既存コンポーネントのスタイル調整
     - 独自コンポーネントの追加

### 主な技術スタック

- **Storybook**: UIコンポーネントの開発・ドキュメント化
- **11ty**: 静的サイトジェネレーター
- **Nunjucks（simple-nunjucks-loader）**: コンポーネントの定義とHTML出力処理を行うテンプレートエンジンとして利用
- **TypeScript**: メインスクリプト (`main.js`) に使用
- **SCSS**: スタイル記述用
